# Code of conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) v2.1.

Short version: be respectful, constructive, and professional. Harassment, discrimination, and personal attacks are not tolerated.

Report violations via GitHub's private reporting features or by contacting the maintainer.

Full text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
